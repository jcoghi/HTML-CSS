Outros comandos

Links

Aqui criamos links de acesso rápido. Foi utilizado o comando < a href ="URL do site">; < /a>

Imagens

nesta seção, colocamos uma imgagem apenas para deixar o site bonitinho. Para colocar a imagem, utilizamos o comando < img src="pasta/imagem.extensão"> ou < img src="URL da imagem"

Divisões da páginas

Utilizamos o div para dividir e organizar melhor o nosso site. Podemos nomear as divs, dessa meneira podemos estilizar cada parte do site de maneira mais eficiente.

Tabelas

Para a criação de tabelas, utilizamos os comandos:
Comando Descrição
Table Cria a estrutura externa da tabela
Tr Cria as linhas da tabela
Td Cria as colunas
Thead Cria o cabeçalho da tabela já deixando texto entralizado e em negrito
Tbody Cria o corpo da tabela.

Comentários

Para colocar um comentário no código, podemos utilizar < !-- antes do quer comentar e --> depois do quer comentar

Formulário

Podemos colocar campos de formuário utilizando o comando input type. A tabela abaixo mostra alguns exemplo de formulários que podemos colocar no site.
Comando Descição Formulário
input type="texto" Cria campo para digitação de texto
input type="password" Cria campo para digitação de texto oculto
input type="button" Cria campo para digitação de texto oculto
input type="submit" Cria campo para digitação de texto oculto
